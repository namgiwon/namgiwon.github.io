1. express-generator 설치

```bash
npm i -g express-generator
```

2. express 프로젝트 생성. --view 옵션은 html 템플릿 엔진이름임(예 : pug, ejs)

```bash
express project-name --view=pug
```

3. 프로젝트로 이동 및 초기화

```bash
cd project-name & npm install
```

4. 실행

```bash
SET DEBUG=project-name:* & npm start
```
